# Python Introduction

## What is Python?

Python is a programming language used to give instructions to computers.

Just like we speak English to humans, we use programming languages to speak with computers.

Python is one of the easiest programming languages for beginners because:

- simple syntax
- easy to read
- less complicated than many other languages
- powerful for real-world applications

## Why Learn Python First?

Python helps beginners focus on logic instead of difficult syntax.

Example:

### Python

```python
print("Hello World")
```

Other languages may require much more code.

This makes Python the best language to start problem solving.

---

## How Python Works

We write code → Python reads it → Computer performs the task

Example:

```python
print("I am learning Python")
```

### Output

```text
I am learning Python
```

---

## Python is Case Sensitive

This means:

```python
name
```

and

```python
Name
```

are different.

Python treats them as separate things.

---

## Important Rule

Computers are very obedient but very strict.

If your instruction is wrong, even small mistakes matter.

That is why syntax is important.

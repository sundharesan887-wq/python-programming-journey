# Built-in Functions in Python

Python gives us many ready-made tools.

We do NOT need to build everything from scratch.

These ready-made tools are called:

# Built-in Functions

They save:

- time
- effort
- code

Very important for beginners.

---

# 🧰 Simple Story

Imagine you want to cut paper.

You can:

build scissors from iron ❌

or

use ready-made scissors ✅

Smart people use ready-made tools.

Python works the same way.

Built-in functions are ready-made tools.

---

# 🧠 What is a Function?

A function is something that:

> does a job

Example:

```python
print()
```

Its job is:

> show output

Simple.

---

# 📚 Common Beginner Built-in Functions

Very important ones:

- `print()`
- `input()`
- `type()`
- `len()`
- `int()`
- `float()`
- `str()`
- `max()`
- `min()`
- `sum()`
- `range()`

We already know some.

Now we understand them properly.

---

# 1️⃣ print()

Used to:

# show output

---

# 💻 Example

```python
print("Hello Python")
```

### Output

```text
Hello Python
```

Python speaks.

---

# 2️⃣ input()

Used to:

# take input from user

---

# 💻 Example

```python
name = input("Enter name: ")
```

Python listens.

---

# 3️⃣ type()

Used to:

# check data type

---

# 💻 Example

```python
age = 21
print(type(age))
```

### Output

```text
<class 'int'>
```

Very useful for debugging.

---

# 4️⃣ len()

Used to:

# count length

Mostly for:

- strings
- lists

---

# 💻 Example

```python
name = "Sundhar"

print(len(name))
```

### Output

```text
7
```

Because there are 7 letters.

---

# 5️⃣ int(), float(), str()

Used for:

# type casting

---

# 💻 Example

```python
num = int("10")
price = float("99.5")
age = str(21)
```

Very important.

---

# 6️⃣ max()

Used to find:

# biggest value

---

# 💻 Example

```python
print(max(10, 25, 7))
```

### Output

```text
25
```

---

# 7️⃣ min()

Used to find:

# smallest value

---

# 💻 Example

```python
print(min(10, 25, 7))
```

### Output

```text
7
```

---

# 8️⃣ sum()

Used to find:

# total addition

---

# 💻 Example

```python
numbers = [10, 20, 30]

print(sum(numbers))
```

### Output

```text
60
```

Very useful later.

---

# 9️⃣ range()

Used in:

# loops

---

# 💻 Example

```python
for i in range(5):
    print(i)
```

### Output

```text
0
1
2
3
4
```

Very important.

---

# 🖥 Example Program

```python
name = input("Enter your name: ")

print("Length:", len(name))
print("Type:", type(name))
```

Simple and powerful.

---

Without built-in functions,

programming becomes slow.

With them,

Python becomes powerful.

Professionals use them every day.

You should too.

